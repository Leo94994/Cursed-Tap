<h2>Про что игра?</h2>
Игра в жанре bullet-hell. Нужно просто уворачиваться от всех объектов на экране, при касании любого объекта игра начивается заново.

<h2>На чем написана?</h2>
Игра написана на Python3 с использованием Pygame.
